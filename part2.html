The Internet- Wires, Cables, Wifi Part 2

  • The Internet basically works on binary; it moves around binary.
  • “The Internet: A tangible physical system made to move information”.
  • The internet is made up of Bits, and it is either 0s and 1s. It is called Binary Code. We send bits through electricity, light, and radio
  • Bandwidth is the maximum transmission capacity and that is measured with bitrate.
  • Bitrate is the number of bits per second that a system can send.
  • Latency is the time that it takes for the sender to send the message to the reciever.
  • The benefits of using electricity would be that it is cheap, but there is a signal loss.
  • The benefits for light would that it is fast and no signal loss, but it is expensive
  • The benefits of radio is that it is wireless, but there is signal loss.
