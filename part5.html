HTTP stands for HyperText Transfer Protocol. It is the language used to communicate between web browsers and servers.

If the request was ever intercepted, you get a “get” request which is just “get” and then the document.

When you enter information you create a post request. The server sees and it allows into the website and also sends an invisible piece of information called cookies which allows the websites to remember who you are. The cookie data is an ID card for tumblr and an ID number that identifies you as you.

The internet is completely open, the connections are shared, and the information is sent in plain text. Safe websites present hackers by communicating on a secure channel.

HTML stands for Hyper Text Markup Language. The text of a webpage is connected to the html, but other parts like images and videos are separate parts with unique urls.

If the page has a lot of images, it requires more http requests, causing the pages to load slower.
